# 麻將深度學習：動作空間 & 特徵空間參考文件

本文檔整理自兩個關鍵原始碼檔案：
- **動作編碼**：`mjx/include/mjx/internal/action.cpp`（`Action::Encode()`）
- **特徵定義**：`convert/convert_mjson_to_features.py`（`FeatureExtractor`）

---

## 1. 動作空間 (Action Space)：181 維 (編碼 0~180)

全部動作編碼定義於 `Action::Encode()`，由 8 位元無符號整數 (`std::uint8_t`) 表示。

### 1.1 DISCARD（手切）：0~36

| 編碼 | 牌 | 說明 |
|------|-----|------|
| 0~33 | m1 ~ rd | 34 種普通牌 |
| 34 | 赤5m | 赤ドラ入りの5マン |
| 35 | 赤5p | 赤ドラ入りの5ピン |
| 36 | 赤5s | 赤ドラ入りの5ソウ |

### 1.2 TSUMOGIRI（摸切）：37~73

| 編碼 | 牌 | 說明 |
|------|-----|------|
| 37~70 | m1 ~ rd | 34 種普通牌 (+37 offset) |
| 71 | 赤5m | 赤5m 摸切 |
| 72 | 赤5p | 赤5p 摸切 |
| 73 | 赤5s | 赤5s 摸切 |

**含赤牌 (95~103)**：

| 96 | m4 | 赤5m 相關（m456, 含赤） |
| 97 | m5 (赤) | 赤5m 相關（m567, 含赤） |
| 98 | p3 | 赤5p 相關（p345, 含赤） |
| 99 | p4 | 赤5p 相關（p456, 含赤） |
| 100 | p5 (赤) | 赤5p 相關（p567, 含赤） |

## 2. 特徵空間 (Feature Space)：1380 維

特徵由兩部分拼接而成：**空間特徵** (1360 維) + **標量特徵** (20 維)。

### 2.1 空間特徵 (Spatial Features)：40 通道 × 34 牌種 = 1360 維

資料類型：`np.uint8`

每通道代表一種牌在特定位置的有無（one-hot 風格），通道中每個列對應 TileType 的 34 種牌 (m1~rd)。

| 通道 | 內容 | 說明 |
|------|------|------|
| **0~3** | 自家手牌 | 每張牌的持有張數對應不同通道（0 通道標記持有 1 張、1 通道標記持有 2 張…之類）|
| **4~7** | 自家副露（最多 4 組） | 已亮出的吃/碰/槓牌 |
| **8~11** | 下家副露 | 右方玩家的副露 |
| **12~15** | 對家副露 | 對面玩家的副露 |
| **16~19** | 上家副露 | 左方玩家的副露 |
| **20~35** | 各家牌河 | 4 名玩家 × 4 組，每組最多 6 張捨牌 |
| **36~39** | 寶牌指示牌 | 最多 4 張表寶牌指示牌 |

> **牌河分配細節**：
> - 通道 20~23：自家牌河（分段：0~5, 6~11, 12~17, 18~23 張）
> - 通道 24~27：下家牌河
> - 通道 28~31：對家牌河
> - 通道 32~35：上家牌河

### 2.2 標量特徵 (Scalar Features)：20 維

資料類型：`np.float32`

| 索引 | 內容 | 歸一化方式 |
|------|------|------------|
| 0 | 自家分數 | `(score + 50000) / 100000` |
| 1 | 下家分數 | 同上 |
| 2 | 對家分數 | 同上 |
| 3 | 上家分數 | 同上 |
| 4 | 剩餘牆牌比例 | `remaining_wall / 70` |
| 5 | 場風＝東 | one-hot：東風戰為 1，否則 0 |
| 6 | 場風＝南 | one-hot：南風戰為 1，否則 0 |
| 7 | 局 1 (東一局) | one-hot |
| 8 | 局 2 (東二局) | one-hot |
| 9 | 局 3 (東三局) | one-hot |
| 10 | 局 4 (東四局) | one-hot |
| 11 | 本場數 (honba) | `honba / 30`（截斷至 1.0） |
| 12 | 立直棒數 (kyotaku) | `kyotaku / 4`（截斷至 1.0） |
| 13 | 自家立直狀態 | 0 或 1 |
| 14 | 下家立直狀態 | 0 或 1 |
| 15 | 對家立直狀態 | 0 或 1 |
| 16 | 上家立直狀態 | 0 或 1 |
| 17 | 自家為莊家 | one-hot |
| 18 | 下家為莊家 | one-hot |
| 19 | 對家為莊家 | one-hot |
